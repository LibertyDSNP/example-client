# About

Simple server for uploading and storing files in public directory.

## Quick Start

```bash
npm install

npm run dev
```

### Run Test

```js
npm run dev
npm run test
```

### API
To upload a file

```js
 /upload?filename=913a707a9eca3f46.parquet
```

To download a file from public

```js
http://localhost:8080/913a707a9eca3f46.parquet
```
